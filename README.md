# Belgian Impact Tariff for Home Assistant

Custom component to calculate Belgian Impact tariff distribution (PIC / MEDIUM / ECO) with support for:

- GRD transport pricing
- Energy compensation (pre 31/12/2023)
- Dynamic pricing
- Import/export tracking

Work in progress.
